# Symbol Extraction

## ADDED Requirements

### Requirement: Java extraction

#### Scenario: Java file dispatch

- **WHEN** a source file has extension `.java`
- **THEN** it SHALL be dispatched to the Java symbol extractor

### Requirement: Kotlin extraction

#### Scenario: Kotlin file dispatch

- **WHEN** a source file has extension `.kt`
- **THEN** it SHALL be dispatched to the Kotlin symbol extractor
